# composition api

设计动机

  • options API
    • 包含一个描述组件的选项（data、methods、props等）的对象
    • options Api 开发复杂组件，同一个功能逻辑的代码被拆分到不同选项
  • composition API
    • Vue.js 3.0 新增的一组api
    • 一组基于函数的api
    • 可以更灵活的组织组件的逻辑

# 响应式系统的升级

  • vue.js 2.x中响应式系统的核心 defineProperty
  • vue.js 3.0中使用proxy 对象重写响应式系统
    • 可以监听动态新增的属性
    • 可以监听删除的属性
    • 可以监听数组的索引和length属性

# 编译优化

  • vue2.0中通过标记静态根节点，优化diff的过程
  • vue3.0中标记和提升所有静态根节点，diff的时候只需要对比动态节点内容
    • fragments
    • 静态提升
    • patch flag
    • 缓存事件处理函数
